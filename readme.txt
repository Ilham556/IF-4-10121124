1. Persiapkan Environment:
Pastikan Anda memiliki Python terinstal di komputer Anda. Jika belum, unduh dan instal Python dari https://www.python.org/.
2. Clone Repository:
git clone <URL_REPOSITORY> atau download manual 
3. Install Dependencies:
Masuk ke direktori aplikasi dan instal semua modul yang diperlukan yang terdaftar dalam file requirements.txt.
melalu cmd jalankan perintah ini:
cd <lokasi_folder>
pip install -r requirements.txt
4.Import Database:
cmd : source path/to/your/file.sql;
atau 
bisa mengguanak phpmyadmin import db
